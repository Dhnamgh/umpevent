# 📊 UMP Event Dashboard

Dashboard quản lý sự kiện UMP sử dụng Streamlit và dữ liệu từ Google Sheets.

## 🚀 Tính năng
- Hiển thị danh sách sự kiện
- Dashboard KPI
- Biểu đồ Gantt timeline
- Lọc theo đơn vị

## 🛠️ Công nghệ
Python, Streamlit, Pandas, Plotly

## ⚙️ Chạy app
bash
pip install -r requirements.txt
streamlit run app.py
